A static multi-page website for a game store — one of my first frontend projects.  

---

## Features

- **Multi-page layout** – main page, catalog, support  
- **Static content** – no backend, all data is hardcoded  
- **Responsive-ready** – basic layout with images and navigation  
- **Vanilla JS** – simple interactivity without frameworks

---

## Tech Stack

- **Language:** HTML5, CSS3, JavaScript (ES5/ES6)  
- **Assets:** Custom images and icons  
- **Tools:** Git, VS Code
